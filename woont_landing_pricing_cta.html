FAQ

자주 묻는 질문

점집이나 무속 서비스와 어떻게 다른가요? +
운트는 전통 명리학의 오행·격국·용신 분석 체계를 기반으로 하되, 모든 결과를 현대 심리학·직무 데이터와 매핑하여 표현합니다. "역마살"을 "글로벌 활동형"으로, "상관"을 "크리에이티브 전문가형"으로 재해석하는 방식입니다. 점괘나 신점이 아닌, 데이터 기반 커리어 가이드입니다.
태어난 시간을 모르면 어떻게 하나요? +
시주(時柱) 없이도 분석이 가능합니다. 년·월·일만으로도 사주 6자 분석이 진행되며, 오행 분포와 격국은 충분히 도출됩니다. 다만 시주까지 입력하면 더욱 정밀한 분석이 이루어집니다.
분석 결과를 다시 볼 수 있나요? +
PRO 플랜 기준으로 분석 결과는 마이페이지에 저장되며 언제든 재열람 가능합니다. PDF 리포트는 다운로드 후 영구 보관할 수 있습니다. 무료 플랜의 경우 당일 세션에서만 확인 가능합니다.
개인정보는 어떻게 보호되나요? +
입력된 생년월일시·이름은 분석 목적으로만 사용되며 제3자에게 판매·제공되지 않습니다. 분석 완료 후 원하시면 즉시 데이터를 삭제할 수 있습니다. 개인정보 처리 방침은 하단 링크에서 확인하세요.
전문가 상담은 어떻게 진행되나요? +
앱 내 상담소에서 전문가를 선택한 뒤 일정을 예약합니다. 상담 전 운트가 자동으로 사주 분석 요약본을 전문가에게 전달하므로 처음부터 설명할 필요 없이 바로 심층 상담이 진행됩니다. 화상·텍스트 모두 지원합니다.
